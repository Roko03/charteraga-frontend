## How to Use

Clone the repository

```bash
git clone [https://github.com/Roko03/react-quiz-app.git](https://github.com/Roko03/charteraga-frontend.git)
cd charteraga-frontend
```

Installation

```bash
npm i
#or
npm install
```

Running the Project

```bash
npm run dev
```
